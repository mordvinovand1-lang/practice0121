# Сумма отрицательных элементов между минимальным и максимальным в массиве

Эта программа находит сумму отрицательных элементов, расположенных между минимальным и максимальным элементами в одномерном массиве.

## Описание задачи

Дан одномерный массив A размерности N. Требуется найти сумму отрицательных элементов, расположенных между максимальным и минимальным элементами массива (не включая сами эти элементы).

## Функциональность

- Генерация массива случайных целых чисел от -10 до 10
- Поиск индексов минимального и максимального элементов
- Определение интервала между ними (исключая граничные элементы)
- Подсчет суммы только отрицательных чисел в этом интервале
- Обработка граничных случаев:
  - Когда между элементами нет промежутка
  - Когда в интервале нет отрицательных элементов
  - При некорректном вводе размера массива
